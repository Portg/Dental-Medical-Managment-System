# generate-controller  v1.0
# 生效日期：2026-03-07  |  初始版本

## 变更说明
- 初始版本：基于口腔医疗管理系统 Web Controller 规范创建

---

## 意图
生成符合项目规范的 Web Controller（非 API），
处理 Blade 视图渲染、DataTable 服务端分页、表单提交。

## 上下文加载顺序
1. `@ai-spec/domain/intent.md`（Anti-Goals + 数据敏感级别）
2. `@ai-spec/domain/entities.md`（仅涉及的实体）
3. `@CLAUDE.md`
4. `@TEST-SPEC.md`
5. `routes/web.php`（现有路由，避免冲突）

## 任务描述
生成：Web Controller + Route + Blade View 骨架 + 对应 CSS/JS 文件

## 验收条件
- [ ] Controller 验证使用 `Validator::make()` 模式
- [ ] JSON 响应使用 `['message' => '', 'status' => 1/0, 'data' => []]` 格式
- [ ] DataTable 使用 `DataTables::of($query)` 服务端处理
- [ ] Blade 视图只包含 HTML 结构，CSS/JS 拆分为独立文件
- [ ] CSS 文件放在 `public/css/<page-name>.css`
- [ ] JS 文件放在 `public/include_js/<page_name>.js`
- [ ] JS 中使用 `LanguageManager.trans()` 获取翻译
- [ ] 路由使用 `->middleware('can:permission-slug')` 权限控制
- [ ] Select2 下拉框配置 zh-CN locale
- [ ] 翻译 key 同步添加到 `resources/lang/zh-CN/` 和 `resources/lang/en/`

## 输出格式
1. 《生成决策说明》
2. 代码文件（Controller / Route / Blade / CSS / JS）
3. 翻译文件（zh-CN + en）
